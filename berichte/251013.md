# Ping Cheat Sheet

## Grundbefehl

```bash
ping <ziel>
```

Testet Erreichbarkeit & Antwortzeit.

## Linux/macOS Optionen

* `-c <anzahl>` → Anzahl Pakete
* `-i <sek>` → Intervall
* `-s <bytes>` → Paketgröße
* `-q` → Nur Zusammenfassung

## Windows Optionen

* `-n <anzahl>` → Anzahl Pakete
* `-t` → Endlos-Ping
* `-l <bytes>` → Paketgröße


Ping arbeitet mit ICMP, insbesondere mit Echo Request und Echo Reply

## Shell Script

```sh
#!/usr/bin/env bash

set -u

NET="${1:-192.168}"

for j in {0..255}; do
  for i in {1..254}; do
    ip="$NET.$j.$i"
    {
      if ping -c 1 "$ip" > /dev/null 2>&1; then
        printf '%s\n' "$ip"
      fi
    } &
  done
done

wait

```
