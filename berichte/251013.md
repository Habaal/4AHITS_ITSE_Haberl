# Ping Cheat Sheet

## Grundbefehl

```bash
ping <ziel>
```

Testet Erreichbarkeit & Antwortzeit.

## Linux/macOS Optionen

* `-c <anzahl>` → Anzahl Pakete
* `-i <sek>` → Intervall
* `-s <bytes>` → Paketgröße
* `-q` → Nur Zusammenfassung

## Windows Optionen

* `-n <anzahl>` → Anzahl Pakete
* `-t` → Endlos-Ping
* `-l <bytes>` → Paketgröße
